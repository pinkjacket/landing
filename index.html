Project List

This is a complete list of projects on my github. They're mostly just practice things, and few of them currently have github pages sites set up.

  • hello-world

    The first project at Epicodus. It just says hello in a few languages, and I think it was actually all copy-pasted code to introduce us to the concept of HTML. Done in HTML.

  • goodbye

    Second Epicodus project. This one says goodbye in several languages, the last two of which were actually added by me and a partner. Done in HTML.

  • cookie-recipe

    Third Epicodus project. Since these are listed in chronological order I may stop numbering them. It's a cookie recipe that needed some cleanup for readability. Done in HTML.

  • my-first-webpage

    This actually contains two pages, and involved more hands-on work than the previous projects and the use of images. Done in HTML.

  • new-website

    A list of some music genres and musicians, with pictures. Also has a second page with a list of movies. First project to use CSS, also first to contain a README file. Done in HTML with CSS.

  • resort-website

    A website for a terrible hotel. Practice for branching and merging. Done in HTML with CSS.

  • cupcake_shop

    A page for a cupcake bakery. Practice for using CSS classes to style HTML elements. Done in HTML with CSS.

  • boring-lecture

    A boring lecture composed mostly of lorem ipsum. It's ostensibly about the Hawley Smoot Tariff Act, which is the subject of Ben Stein's lecture in Ferris Bueller's Day Off, chosen by John Hughes because it was the most boring awful adult subject he could think of. Further CSS styling practice. Done in HTML with CSS.

  • band

    A brief page about the Pet Shop Boys, my favorite band, with photos. Intended as float practice, we also ended up incorporating columns a lesson early to keep the pictures and text from overlapping. Done in HTML with CSS.

  • blog

    A small blog mockup. Mostly lorem ipsum, done for practice with things like margins and padding. Also contains a gif that you used to find on a lot of blogs. A tooltip was also attempted but we couldn't quite get it to work. Done in HTML with CSS.

  • petsite

    An animal adoption website done to practice cascading and media queries. Done in HTML with CSS.

  • vacation

    A website advertising vacations to scenic Europa, theoretically-inhabitable ice moon of Jupiter. Done as an introduction to using Bootstrap. Done in HTML with CSS and Bootstrap.

  • wiki

    An attempt to recreate the front page of Wikipedia using Bootstrap. It's mising the nav bar and a few other things, but I think it came out all right for a first try. Done in HTML with CSS and Bootstrap.

  • landing

    The repository for this page, right here. Weird feeling, right? Done in HTML with CSS and Bootstrap.

Three Facts about John Morrissey, America's first Irish mob boss

numbered but in no meaningful order
